# AprilTag 3D Model Tracker

Приложение для трекинга 3D модели с использованием AprilTag маркеров.
Поддерживает USB и CSI камеры, уточнение позы по геометрическим признакам пирамиды.

## 📋 Требования

- Python 3.8+
- OpenCV
- numpy
- pyapriltags
- Для CSI камер: Raspberry Pi с picamera2

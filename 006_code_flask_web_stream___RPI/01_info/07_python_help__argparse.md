# Модуль argparse для обработки аргументов командной строки. 
Как это работает:

    Создание парсера аргументов:
        argparse.ArgumentParser() создает объект для обработки аргументов командной строки
        
        Параметр description задает описание программы, которое показывается в справке

    Добавление аргумента:

        parser.add_argument() добавляет один аргумент командной строки

        В данном случае определен аргумент --config (длинная форма) и его псевдоним -c (короткая форма)

        default='config_rpi.yaml' устанавливает значение по умолчанию

        help=... предоставляет текстовое описание аргумента для справки

    Парсинг аргументов:

        args = parser.parse_args() обрабатывает переданные аргументы

        Результат сохраняется в объект args

    Использование аргумента:

        args.config содержит путь к конфигурационному файлу

        Этот путь передается в функции create_logger(), load_config(), и logger.log_startup_info()

Типы параметров:
В add_argument():

    Позиционные аргументы - не используются здесь

    Опциональные аргументы (используются с префиксами -- или -):

        '--config' - длинная форма (два дефиса)

        '-c' - короткая форма (один дефис)

        По умолчанию являются необязательными

    Параметры метода add_argument():

        default='config_rpi.yaml' - значение по умолчанию (строка)

        help='...' - строка помощи

        Отсутствующий параметр type: значит аргумент автоматически обрабатывается как строка (string)

        Отсутствующий параметр required: значит аргумент необязательный (optional)

Тип возвращаемого значения:

    args.config будет иметь тип string (строка), так как не указан параметр type

    Если аргумент не передан в командной строке, будет использовано значение по умолчанию 'config_rpi.yaml'

Пример использования:
bash

# Без аргументов - используется значение по умолчанию
python script.py

# С короткой формой аргумента
python script.py -c my_config.yaml

# С длинной формой аргумента
python script.py --config custom_config.yaml

# Просмотр справки
python script.py --help

Вывод справки будет содержать:
text

usage: script.py [-h] [--config CONFIG]

Flask Webcam Stream with YAML Configuration

optional arguments:
  -h, --help            show this help message and exit
  --config CONFIG, -c CONFIG
                        Путь к конфигурационному файлу YAML (по умолчанию: config_rpi.yaml)
